# Consistent Hashing Simulation (Python)

This project demonstrates how Consistent Hashing reduces key redistribution compared to modulo-based hashing when nodes are removed.

## Files

- `consistent_hash_ring.py` – Implements consistent hashing
- `simulate.py` – Simulation logic
- `consistent_hash_project.py` – Main runner
- `report.txt` – Explanation of design
- `simulation_log.txt` – Example output logs
- `summary.json` – Summary of simulation

## Run the project

```sh
python consistent_hash_project.py
